Authentication core for SlimPower Framework

##Installation

In terminal:

```sh
composer require matiasnamendola/slimpower-auth-core
```

Or you can add use this as your composer.json:

```json
{
    "require": {
        "matiasnamendola/slimpower-auth-core": "dev-master"
    }
}
```
